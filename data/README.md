# Data

Place datasets here. Dataset files are intentionally ignored by Git.

Expected layout:

```text
data/
  CD1/
    Images/
      CBB/
      CBSD/
      CGM/
      CMD/
      Healthy/
  CD2/
    Images/
      CBSD/
      CMD/
      Healthy/
  CD3/
    Images/
      CBB/
      CMD/
      Healthy/
```

The cleaned scripts use `data/CD*/Images` by default.

Dataset references from the paper:

- CD1: Cassava Leaf Disease Classification, Kaggle.
- CD2: Makerere University Cassava Image Dataset, Harvard Dataverse.
- CD3: Cassava Leaf Disease Dataset, Mendeley Data.
